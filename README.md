# DoubanFM

DoubanFM desktop version built on electron. 

## INSTALLATION

* Windows


* MacOS

## FEATURES

- [x] 基本播放功能
- [x] 进度条拖动
- [x] 频道选择
- [x] 用户登录
- [x] 歌曲收藏
- [x] 歌曲剔除
- [ ] 快捷键
- [ ] 音效设置
- [x] 歌词同步
- [ ] 均衡器调节

## SHORTCUTS
`Space` play/pause

`Ctrl` + `D` never show again

`Ctrl` + `N` next

`Ctrl` + `L` login

`Ctrl` + `U` like/dislike

`Ctrl` + `M` show/hide menu

`Ctrl` + `S` show/hide lrc

`Ctrl` + `Q` exit
